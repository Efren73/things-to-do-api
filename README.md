## How to install our code in your computer?
* Clone the repo 
```shell
git clone https://github.com/Efren73/GoWheelTheWorld-Front.git
```
* Go to the project directory
```shell
cd project
```
* Install dependencies
```shell
npm install
```
* To start the project
```shell
npm run dev
```
